# work-manage
基于springboot的收作业工具
 springboot整合Mybatis，实现工作部署和任务提交。前后端分离。第一次commit只实现了基本功能，未关注权限分离和安全漏洞。和预期系统差很多，有空慢慢更新。

  - 批量导入成员用户。
  - 增加作业。
  - 成员用户提交作业。
  - 下载单项作业压缩包。
